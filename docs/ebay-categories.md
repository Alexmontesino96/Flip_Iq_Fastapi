## Cómo Obtener Categorías via API

### eBay Taxonomy API

```bash
# 1. Obtener token (client credentials)
curl -X POST https://api.ebay.com/identity/v1/oauth2/token \
  -H "Authorization: Basic $(echo -n 'APP_ID:CERT_ID' | base64)" \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "grant_type=client_credentials&scope=https://api.ebay.com/oauth/api_scope"

# 2. Árbol completo (US = tree 0)
curl https://api.ebay.com/commerce/taxonomy/v1/category_tree/0 \
  -H "Authorization: Bearer {token}" \
  -H "Accept-Encoding: gzip"

# 3. Sugerencia por keyword
curl "https://api.ebay.com/commerce/taxonomy/v1/category_tree/0/get_category_suggestions?q=Nike+Air+Jordan" \
  -H "Authorization: Bearer {token}"

# 4. Sub-árbol desde una categoría
curl "https://api.ebay.com/commerce/taxonomy/v1/category_tree/0/get_category_subtree?category_id=15032" \
  -H "Authorization: Bearer {token}"
```

### Auth requerida
- **Scope**: `https://api.ebay.com/oauth/api_scope` (público, sin login de usuario)
- **Grant**: `client_credentials`
- **Credenciales FlipIQ**: `EBAY_APP_ID` + `EBAY_CERT_ID` del `.env`

## Notas

- Los IDs pueden tener variantes (eBay a veces usa IDs diferentes para la misma subcategoría en distintos contextos)
- La Taxonomy API se actualiza periódicamente — consultar `categoryTreeVersion` para detectar cambios
- Para FlipIQ, las ~50 subcategorías listadas arriba cubren >90% del volumen de reselling
- `getCategorySuggestions` es útil para auto-detectar categoría sin depender del LLM
